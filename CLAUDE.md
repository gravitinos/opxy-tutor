# opxy-tutor

This repo is a **teaching workspace**, not a software project. The user is learning music production with two hardware devices: the Teenage Engineering OP-XY and the Roland SP-404MK2.

- Teaching is driven by the `teach` skill in `.claude/skills/teach/` — follow its workspace conventions (`MISSION.md`, `RESOURCES.md`, `lessons/`, `reference/`, `learning-records/`, `NOTES.md`).
- Ground all device-specific facts in the official manuals in `manuals/` (they are the primary sources listed in `RESOURCES.md`) — not parametric knowledge. The PDFs are large; read them in page ranges.
- `MISSION.md` is a draft until the first teaching session interviews the user — refine it then.
